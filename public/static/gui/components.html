图标

可用图标

图标没有用字体图标，而是图片。所以无法设置颜色和font-size，请按常规方法使用图片。

Less代码

采用Less代码循环生成图标样式，增减图标只需修改@icon-names

  • 注意

    要改变图片大小，请使用background-size

  • 
//less
﻿//@icon-names，图标.icon-*的命名规则
@icon-names:
    amplify,beep,clock,computer,
    download,email,exit,feedback,
    home,horn,manager,mobile,
    msg,pencil,position,record,
    reduce,register,report,set,
    user,userinfo,userlog,usermanage;
@icon-count: length(@icon-names);
//@icon-url，图标路径
@icon-url: '../img/icon/';
//循环生成图标
.icon-variant(@i: 1) when (@i =< @icon-count) {
    @name: extract(@icon-names,@i);
    .icon-@{name} {
        background-image: url('@{icon-url}@{name}.png');
    }
    .icon-variant(@i + 1);
}
.icon-variant;
//.icon基本样式，没有设置width和height，用户自定义
.icon {
    display: inline-block;
    background-size:auto 100%;
    background-position:center center;
    background-repeat:no-repeat;
}
//生成的css代码===================================
.icon-amplify {
  background-image: url('../img/icon/amplify.png');
}
...

  • 样式类名介绍

    样式之间可以相互组合，组合成更丰富的效果。

  • 类名 效果
    .menu display:block只是让菜单显示，没有其他效果，所以隐藏的菜单(下拉菜单)一定不要加这个样式
    .menu-vertical 垂直菜单
    .menu-horizontal 水平菜单
    .menu-default 默认样式
    .menu-separate li{border-bottom: 1px solid #fff;}白色分隔线
    .menu-trans li{border-bottom:tansparant;}无分隔线
    .menu-primary 主题色菜单
    .menu-primarydark 深颜色主题色菜单
    .menu-pure 无背景色菜单
    .menu-condensed padding减小，紧缩菜单
    .menu-sub 子集菜单

圆角

提供一些圆角类，可以改变圆角的样式和大小

圆角大小

.radius-*,.metro类，设置圆角大小。.radius(4px) | .radius-lg(6px) | .radius-sm(3px) | .metro(0px)，添加了!important优先级。

  • ﻿﻿<!--Less代码-->
.circle{
    border-radius:@input-height-computed !important;//35px
}
.circle-lg{
    border-radius:@input-height-large-computed !important;//45px
}
.circle-sm{
    border-radius:@input-height-small-computed !important;//30px
}
.metro{
    border-radius:0px !important;//无圆角
}
.radius{
    border-radius:@border-radius-base !important;//4px
}
.radius-lg{
    border-radius:@border-radius-large !important;//6px
}
.radius-sm{
    border-radius:@border-radius-small !important;//3px
}

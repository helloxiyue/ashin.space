说明

修改了bootstrap的源文件，主要向三个方向改动：
一、常用样式兼容到ie7
二、修改了配色和一些样式
三、添加了一些样式
所以，使用方法和bootstrap一模一样，不需要额外的学习成本。下面的说明文档只是展示nano.bootstrap修改了的地方，没有修改的地方，虽然没有展示，但依然保留了原bootstrap的样式，就不重复写文档了。其中有很多是兼容性改写的策略和代码，希望对大家有帮助。

兼容解决方案

bootstrap在ie9以下变形最大的原因是因为设置了*{box-zising:border-box;}和&:after{clear:both}，所以我取消了这两种设置，带来的麻烦是我不能设置@grid-gutter-width和我必须要手动消除浮动。但不要担心啦，你只需要引入nano-bootstrap.js就可以解决兼容问题了，它会在高版本浏览器加上&:after的设置，在低版本浏览器为.row添加消除浮动的子元素.row-clear。另外,由于低版本浏览器不支持媒体查询，所以.col-sm-*和.col-md-*和.col-lg-*在低版本的时候也会被js替换成.col-xs-*

表单

兼容方案

box-sizing:border-box时：outerHeight = height（同理width）
box-sizing:border-box时：outerHeight = line-height + padding-top + padding-bottom + border-top + border-bottom（同理width）
ie9以下不支持box-sizing，使得表单元素变形，解决方案如下

nano-bootstrap.css 高度解决方案

.form-control{//设置高度，不支持box-sizing的浏览器正常显示
        box-sizing:content-box;
        widht:100%;
        height:@input-height-base;//23px
        &.input-lg{
            height:@input-height-large;//23px
        }
        &.input-sm{
            height:@input-height-small;//22px
        }
    }
//浏览器宽度肯定大于1px，下面代码会覆盖上面的设置
﻿@media (min-width:1px){  //支持媒体查询的一般支持box-sizing,通过这种方法进行覆盖
    .form-control{
        box-sizing:border-box;
        height:@input-height-computed;//35px
        &.input-lg{
            height:@input-height-large-computed;//45px
        }
        &.input-sm{
            height:@input-height-small-computed;30px
        }
    }
}

宽度兼容方案，这段代码会在不支持border-box的浏览器中执行。

﻿jk.windowOnlode(function () {//windown.onload执行
    if (!window.matchMedia("(min-width:1px)").matches) {

内联表单

为 <form> 元素添加 .form-inline 类可使其内容左对齐并且表现为 inline-block 级别的控件。只适用于视口（viewport）至少在 768px 宽度时（视口宽度再小的话就会使表单折叠）。

  • 兼容方案

    引入补丁文件nano-bootstrap.patch.css

  • //引入patch
﻿<!--[if lt IE 9]>
    <link type="text/css" rel="stylesheet" href="../../css/nano-bootstrap.patch.css" />
<![endif]-->
//patch中的相关兼容代码
.form-inline {//取消媒体查询
    .form-group {
        display: inline;//inline-block在ie7要换行，所以改成inline
        margin-bottom: 0;
        vertical-align: middle;
    }
    .form-control {
        display: inline;
        width: auto;
        vertical-align: middle;
    }
    .form-control-static {
        display: inline;
    }
    .input-group {
        display: inline-table;
        vertical-align: middle;
        .input-group-addon,
        .input-group-btn,
        .form-control {
            width: auto;
        }
    }
    .input-group > .form-control {
        width: 100%;
    }
    .control-label {
        margin-bottom: 0;
        vertical-align: middle;
    }
    .radio,
    .checkbox {
        display: inline;
        margin-top: 0;
        margin-bottom: 0;
        vertical-align: middle;
        label {
            padding-left: 0;
        }
    }
    .radio input[type="radio"],
    .checkbox input[type="checkbox"] {
        position: relative;
        margin-left: 0;
    }
    .has-feedback .form-control-feedback {
        top: 0;
    }
    .has-tip .form-control-tip {
        top: 0;
    }
}
